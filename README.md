A hands-on learning repository where I explore **Retrieval-Augmented Generation (RAG)** from the ground up through practical implementation.

The goal of this project is to understand every stage of a RAG pipeline by building and experimenting with each component instead of only using high-level frameworks.

---

## 🎯 Objective

This repository documents my journey of learning RAG by implementing and understanding:

- Data Ingestion
- Document Parsing
- Text Splitting
- Embedding Generation
- Vector Databases
- Similarity Search
- Retrieval Strategies
- Prompt Engineering
- Large Language Models (LLMs)
- End-to-End RAG Pipeline
- Advanced RAG Techniques

Rather than treating RAG as a black box, this project focuses on understanding **how and why each component works**.
